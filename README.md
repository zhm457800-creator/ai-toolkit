# 🧭 AI Toolkit Explorer

> 一个精心设计的 AI 工具导航页，用于收集、管理和快速访问你常用的 AI 工具。采用极简设计语言，支持黑白主题一键切换，所有数据保存在浏览器本地。

---

## ✨ 功能特性

### 🎨 视觉设计
- **双主题切换** — 极昼（浅色）与暗黑模式一键切换，默认浅色背景
- **磨砂玻璃效果** — 所有浮动组件（导航栏、卡片、弹窗、面板）均采用 `backdrop-filter` 磨砂玻璃效果
- **3D 卡片交互** — 鼠标悬停时卡片产生 3D 透视倾斜效果
- **响应式布局** — 完美适配桌面端、平板和手机

### 🔧 核心功能
- **工具卡片展示** — 每个工具以卡片形式展示，包含图标、名称、描述、标签、评分、最后使用时间
- **分类筛选** — 支持按标签分类筛选：文本生成、图像生成、代码辅助、音频、视频、效率工具
- **搜索功能** — 实时搜索工具名称、描述和标签
- **收藏系统** — 双击卡片可快速收藏/取消收藏
- **评分系统** — 五星评分，支持在详情面板中修改
- **使用笔记** — 每个工具可记录个人使用心得
- **最后使用时间** — 基于 30 天热力图数据，智能计算最后使用时间

### ✏️ 编辑模式
- **密码保护** — 编辑模式需要密码验证（默认密码：`explorer`）
- **添加工具** — 弹出式添加卡片，支持粘贴或拖拽 URL，自动识别工具信息
- **自定义标签** — 可选择已有标签或添加自定义标签，新标签自动出现在筛选栏
- **删除工具** — 在详情面板中可删除工具
- **草稿管理** — 新添加的工具默认为草稿状态，可手动发布

### 📊 对比功能
- **多工具对比** — 勾选 2~3 个工具进行并排对比
- **对比维度** — 名称、网址、评分、最后使用时间、标签、描述、笔记

### 🌐 数据管理
- **本地存储** — 所有数据保存在 `localStorage`，无需后端
- **自动识别** — 内置 25+ 常用 AI 工具的域名映射，自动获取名称和标签
- **Favicon 图标** — 自动获取网站真实图标，无需手动配置

## 🚀 快速开始

### 方式一：直接打开
下载 `plan1-enhanced.html` 文件，双击用浏览器打开即可。

### 方式二：本地服务器
```bash
# 使用 Python
python3 -m http.server 8080

# 或使用 Node.js
npx serve .
```
然后访问 `http://localhost:8080/plan1-enhanced.html`

### 方式三：部署到 GitHub Pages
1. 将 `plan1-enhanced.html` 推送到 GitHub 仓库
2. 仓库 Settings → Pages → Source 选 `main` 分支
3. 访问 `https://你的用户名.github.io/仓库名/plan1-enhanced.html`

### 设计规范

#### 配色系统

| 属性 | 暗黑模式 | 极昼模式 |
|------|---------|---------|
| 背景 | `#1A1A1B` | `#F4F1ED` |
| 文字 | `#F4F1ED` | `#1A1A1B` |
| 表面 | `rgba(40,38,37,0.65)` | `rgba(255,255,255,0.45)` |
| 边框 | `rgba(244,241,237,0.08)` | `rgba(26,26,27,0.08)` |
| 强调色 | `#d4a5a5` | `#b07070` |

两种模式完美互为反色，确保一致的视觉体验。

#### 磨砂玻璃效果
所有浮动组件统一使用 `backdrop-filter: blur(16~24px) saturate(1.2)` 实现磨砂玻璃效果，配合半透明背景和微妙的顶部高光线（`inset box-shadow`），营造层次感。

#### 标签分类系统
| 英文标签 | 中文标签 | 说明 |
|---------|---------|------|
| `text` | 文本生成 | ChatGPT、Claude、Kimi 等 |
| `image` | 图像生成 | Midjourney、DALL·E、Stable Diffusion 等 |
| `code` | 代码辅助 | GitHub Copilot、Cursor、Windsurf 等 |
| `audio` | 音频 | ElevenLabs、Suno、Whisper 等 |
| `video` | 视频 | Runway、Kling AI、Luma AI 等 |
| `productivity` | 效率工具 | Notion AI、Perplexity、Gamma 等 |

支持中英文双语标签，用户还可自定义新标签。

## 📖 使用指南

### 基础操作

| 操作 | 说明 |
|------|------|
| **浏览工具** | 直接滚动查看所有工具卡片 |
| **查看详情** | 单击卡片打开侧滑详情面板 |
| **收藏工具** | 双击卡片快速收藏/取消 |
| **搜索工具** | 在搜索框输入关键词实时筛选 |
| **分类筛选** | 点击标签按钮按类别过滤 |
| **切换主题** | 点击导航栏 ☀/☾ 按钮切换黑白背景 |

### 编辑模式

1. 点击导航栏 🔒 按钮或右侧「添加工具」按钮
2. 输入密码（默认：`explorer`）
3. 进入编辑模式后可：
   - 点击「添加工具」按钮 → 弹出添加卡片 → 粘贴 URL → 确认添加
   - 单击卡片 → 在详情面板中修改评分、笔记、标签
   - 勾选卡片左上角复选框 → 选择 2~3 个工具进行对比
   - 点击导航栏 🔓 按钮退出编辑模式

### 添加新工具

1. 解锁编辑模式
2. 点击「添加工具」按钮
3. 在弹出卡片中粘贴或拖拽 URL
4. 系统自动识别工具名称、图标和标签
5. 点击「添加到收藏」确认
6. 新工具以草稿状态添加，可在详情面板中编辑后发布

---

## 🔒 安全说明

- 编辑模式通过密码保护，防止误操作
- 默认密码为 `explorer`，可在源码中修改 `CORRECT_HASH` 变量来更改
- 密码使用 SHA-256 哈希存储，不以明文保存在代码中
- 连续 5 次密码错误将暂时锁定输入
